== 나만의 커뮤니티 기획서 ==

1. 서비스 개요
- 서비스명: Mat_Mut
- 한 줄 소개: 우리 동네 진짜 맛집을 발견하고 공유하는 맛집 정보 공유 커뮤니티
- 타겟 사용자: 맛집 탐방을 즐기는 사용자, 새로 이사한 동네의 맛집을 찾는 사용자, 배달/외식 전 리뷰를 참고하고 싶은 사용자

2. 커뮤니티 주제
- 맛집 정보 공유

3. 디자인 스타일
- 사이트 이름: Mat_Mut
- 색상 테마
    - 메인 컬러: 주황색 (포인트 강조, 버튼)
    - 서브 컬러: 노란색 (배경, 카드)
    - 포인트 컬러: 연두색 (좋아요, 성공 메시지 등 긍정적 액션)
- 분위기: 재미있고 귀여운 분위기 (둥근 모서리, 일러스트/이모지 활용)
- 폰트: 가독성 좋은 라운드 고딕 계열 추천 (예: Cafe24 Ssurround, 카카오 small sans 등)

4. 핵심 기능 아이디어
- 맛집 평점 (별점 5점 만점)
- 한 줄 평
- 사진 업로드 (다중 업로드 지원)
- 즐겨찾기(북마크)
- 검색 기능: 가게 이름, 지역, 음식 종류 기준 검색
- 정렬 기능: 인기순 / 최신순 / 평점순
- 카테고리 필터: 음식 종류별, 가격대별

5. 회원 등급 및 권한
- 비회원: 게시물 열람만 가능
- 일반 회원: 게시물/댓글 작성, 평점 등록, 즐겨찾기, 신고 접수
- 관리자: 신고된 게시물/댓글 검토 및 삭제, 회원 제재

6. 페이지별 특징

6-1. 메인 페이지
- 실시간 가입자가 가장 많이 찾아보는 인기 맛집 5 ~ 10곳
- 왼쪽 사이드 바를 상하로 나누어 음식 종류와 가게 위치 검색
- 상단 검색창 (가게 이름/지역 통합 검색)

6-2. 로그인 페이지
- 이메일 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼
- 회원가입 페이지 이동 버튼
- 비밀번호 찾기/재설정 페이지 이동 버튼

6-3. 회원가입 페이지
- 이메일 입력 필드
- 이메일 중복 검사 버튼
- 비밀번호 입력 필드
- 비밀번호 확인 입력 필드
- 닉네임(이름)
- 이용약관/개인정보처리방침 동의 체크박스
- 회원가입 버튼

6-4. 게시물 목록
- 게시물 정보
    - 가게 상호명
    - 가게 위치
    - 음식 이미지
    - 음식 주재료
    - 가격
    - 평점
    - 댓글 수
- 왼쪽 사이드 바에 요리 종류별 카테고리
- 정렬 옵션 (인기순/최신순/평점순) 및 페이지네이션(또는 무한 스크롤)

6-5. 게시물 상세
- 가게 상호명
- 가게 위치 (지도 연동 권장)
- 음식 이미지 (여러 장인 경우 슬라이드)
- 음식 주재료
- 가격
- 평점
- 한 줄 평
- 즐겨찾기 버튼
- 댓글 영역
    - 댓글 작성자
    - 댓글 내용
    - 댓글 작성 시간
    - 작성자가 부여한 평점

6-6. 게시물 작성
- 제목
- 가게 상호명 / 위치
- 이미지 첨부 (다중)
- 상세 내용
- 음식 주재료, 가격, 평점 입력

6-7. 게시물 수정
- 제목
- 이미지 첨부
- 상세 내용
- 기존 작성 내용 불러오기 (작성자 본인만 접근 가능)

6-8. 마이페이지
- 내가 작성한 글
- 내가 작성한 댓글
- 즐겨찾기
- 최근 본 글
- 프로필 수정 (닉네임, 비밀번호 변경)
- 회원 탈퇴

6-9. 관리자 페이지 (신규)
- 신고 접수 목록 조회
- 신고된 게시물/댓글 처리 (경고/삭제)
- 회원 관리 (제재/차단)

7. 댓글 기능
- 대댓글
- 댓글 좋아요
- 댓글 상단 고정
- 댓글 신고

8. 신고 및 제재 정책 (신규)
- 신고 사유 선택형 입력 (욕설, 광고/스팸, 허위 정보, 기타)
- 신고 누적 시 자동 숨김 처리 기준 (예: 5회 누적 시 임시 비공개)
- 관리자 최종 검토 후 삭제/복구 결정

9. 기술 스택 제안 (추후 확정 필요)
- 프론트엔드: React (Vite)
- 백엔드/DB: Supabase (Auth, Database, Storage)
- 배포: GitHub Pages 또는 Vercel

10. 개발 우선순위 (MVP)
- 1단계(MVP): 회원가입/로그인, 게시물 목록/상세/작성, 댓글, 평점
- 2단계: 검색/정렬/카테고리 필터, 즐겨찾기, 마이페이지
- 3단계: 대댓글, 댓글 좋아요/신고, 관리자 페이지, 신고 자동 처리
